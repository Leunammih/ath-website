# Annika Tara Website

Static website for life coach **Annika Tara** (client project, separate from MindBodyWorkFlow).

## Structure
- `site/` — the website: `index.html`, `styles.css`, `site.js`, `assets/`, `README.md`
- `Website AT reference elements/` — source images: Annika's portrait (HEIC), logo PNG, plant/section graphics. Optimize before using; never deploy originals.

## Conventions
- No build step — plain HTML/CSS/JS. Preview via the Claude Preview dev server.
- Keep brand assets (logo, portrait) exactly as provided — don't regenerate her likeness or logo with AI.
- Update STATUS.md at the end of every working session.
